# JSS-hackathon

JSS Hackathon - EcoAQI Hub
Dual-purpose platform combining real-time AQI monitoring with interactive maps and an e-commerce site for environmental contributions with AI-verified photo uploads and rewards.

🚀 What We Built
Users check live Air Quality Index (AQI) for their locality using interactive maps powered by AQICN API. They can also upload environment-related photos (verified by Gemini AI image recognition), contribute to green initiatives, and claim e-commerce rewards—all in one seamless platform with an environmental chatbot.

✨ Key Features
Live AQI Maps: Real-time pollution levels by location using AQICN API

Photo Contributions: Upload environment photos validated by Gemini AI image recognition

E-commerce Rewards: Claim rewards for verified green contributions

Environment Chatbot: General Q&A about pollution, sustainability, and eco-tips
